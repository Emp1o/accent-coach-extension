# Accent Coach FREE (без OpenAI)

Это clean-версия проекта на базе исходного архива.
Структура, экраны, карточки, стили и иконки сохранены.

## Что изменено
- OpenAI и обязательный backend убраны из пользовательского пути.
- Расстановка ударений и запятых работает локально внутри расширения.
- В проект добавлен `landing/` для сайта с кнопкой скачивания и Яндекс Метрикой.
- Backend остаётся в архиве как необязательная часть для будущих экспериментов, но пользователю он не нужен.

## Как распространять
1. Залей проект на GitHub.
2. Создай Release и приложи ZIP.
3. На лендинге поставь ссылку на GitHub Release.
4. Подключи Яндекс Метрику.
5. Пользователь скачивает ZIP, распаковывает его и загружает папку `extension` через Developer mode.


## Новый модуль тренировки
- Добавлена отдельная страница `training.html`.
- Тренировка запускается из `Все карточки` или из popup.
- По умолчанию сессия = 10 карточек.
